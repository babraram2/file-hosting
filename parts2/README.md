## 🔧 How to Reassemble

### Linux/macOS:
```bash
# Download all parts2, then:
cat filename.zip.* > filename.zip
unzip filename.zip
```

### Windows (Command Prompt):
```cmd
copy /b filename.zip.001 + filename.zip.002 + filename.zip.003 filename.zip
```
